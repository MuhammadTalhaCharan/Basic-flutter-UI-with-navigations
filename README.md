# Flutter Login App

## Description
A simple Flutter application demonstrating:

- Login Screen UI
- Email Validation
- Password Validation
- Navigation between Screens

## Features
- Beautiful Login UI
- Email format validation
- Password validation
- Home Screen navigation

## Technologies
- Flutter
- Dart

## How to Run

1. Clone repository
2. Open project in VS Code/Android Studio
3. Run:

flutter pub get

flutter run
